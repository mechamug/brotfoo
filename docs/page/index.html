    The same as a regular Mandelbrot set, but instead of the boring solid color, the same technique as for some domain coloring approaches is used, with a color index being chosen on the interval $[-\pi, \pi]$, scaled to the range of the maximum iterations chosen, i.e. the escape bound.

    A really simple way to do domain coloring in Python. Domain coloring of the function $f(x) = \frac{sinc(z+i)}{tan(z-i)}$: import numpy as np import matplotlib.pyplot as plt x,y = np.ogrid[-10:6:2560j, -10:10:1600j] f = lambda z: np.sinc(z+i) / np.tan(z-i) plt.imsave('graph.png', np.angle(f((x - 1j*y).T)), cmap='twilight') Another Example
